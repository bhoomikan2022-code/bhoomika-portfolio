# N Bhoomika Portfolio

A premium personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. Designed for recruiters and modern AI/cloud/data engineering hiring teams.

## About

This project showcases the profile of N Bhoomika, an Integrated M.Tech Software Engineering student at VIT Vellore specializing in AI, cloud, and data engineering. The site includes the hero section, about summary, skills, internship experience, featured projects, certifications, and contact form.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Features

- Modern glassmorphism dark theme
- Fully responsive layout
- Smooth entrance and hover animations
- Floating particle background
- Sticky transparent navbar
- Scroll progress indicator
- Project filtering and detail modal
- Contact form
- Production-ready build configuration

## Projects

- **SecureVault** – Secure cloud file storage system with AES-256 encryption, chunking, compression, and SHA deduplication.
- **RFP Insight** – RFP and tender analysis platform with AI-powered summarization and dashboard insights.
- **RecipeGenie** – AI-powered recipe finder with ingredient-based search and intelligent matching.
- **SpendWise** – Intelligent expense tracker with budget insights and MySQL storage.
- **CareSync** – Clinical appointment and record management system built with Java and MySQL.
- **ForecastIQ** – Business analytics and forecasting system using Python, Pandas, and Scikit-learn.

## Getting Started

### Install dependencies

```bash
cd "C:\Users\Bhoomika N\portfolio-site"
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Deployment

This project is ready for deployment on Vercel. The production build output is generated to the `dist/` folder, which Vercel can serve directly.
